# Nyx Anvil Notes

> Brainstorm, ide, dan gambaran jangka panjang.

## Vision

Nyx Anvil bukan sekadar obfuscator.

Tujuan jangka panjang adalah menjadi security hardening pipeline yang dapat diterapkan pada berbagai jenis artifact tanpa terikat pada engine tertentu.
